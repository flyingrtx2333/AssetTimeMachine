# AssetTimeMachine 商店截图草稿

本目录保存一组用于 App Store 页面设计的基础素材：

- `01-home-dashboard.png` 首页总览
- `02-records-editor.png` 记录页
- `03-time-machine.png` 时光机页
- `04-backtest.png` 回测页
- `captions.zh-CN.md` 中文配文建议
- `captions.zh-CN.json` 结构化配文
- `sample-import.json` 截图用合成样例数据

说明：
- 设备为 **iPhone 17 Pro Max Simulator**
- 使用的是 **合成样例数据**，不是用户真实资产
- 状态栏已统一为 **9:41**，便于后续商店图设计
